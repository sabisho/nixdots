# Visual Mode
    1. `o` this moves the cursor in visual mode from one end to other.
    2. `g-<C-a>` this increases a list of number and add +1.
    3. `gv` this reselects the selection after some operation.

# Command Mode
    1. `%y` this yanks the whole buffer into clipboard.
    2. `sort` this sorts the selection in ascending order.
    3.        this sorts the selection in descending order.
